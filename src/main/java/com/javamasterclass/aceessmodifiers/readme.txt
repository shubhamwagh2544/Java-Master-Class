###
access modifiers - 4
- private
- protected
- default
- public

private - the strongest form of encapsulation
public - the weakest form of encapsulation

private -> visibility same class only -> if inheritance :-> no override
protected -> visibility same class and inherited classes only -> if inheritance :-> can override -> but can't narrow down scope
default -> visibility in same package -> if inheritance :-> can override -> but can't narrow down scope
public -> visibility everywhere -> if inheritance :-> can override -> but can't narrow down scope
